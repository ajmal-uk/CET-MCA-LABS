JavaScript provides many methods to work with strings. Here are a few commonly used ones:

  • length - Returns the length of the string.
  • toUpperCase() - Converts string to uppercase.
  • toLowerCase() - Converts string to lowercase.
  • charAt(index) - Returns the character at a given position.
  • substring(start, end) - Extracts part of a string.
  • indexOf(value) - Finds the position of the first occurrence.
  • replace(search, replace) - Replaces part of the string.

The Math object has many useful methods for mathematical calculations:

  • Math.round(x) - Rounds to the nearest integer.
  • Math.floor(x) - Rounds down.
  • Math.ceil(x) - Rounds up.
  • Math.random() - Returns a random number between 0 and 1.
  • Math.max(...values) - Returns the largest value.
  • Math.min(...values) - Returns the smallest value.
  • Math.sqrt(x) - Returns square root.
  • Math.pow(x, y) - Returns x to the power of y.
